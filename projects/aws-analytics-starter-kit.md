# Project: AWS Analytics Starter Kit

## Purpose

Create a reusable SMB-friendly analytics pattern for customers that are currently using spreadsheets and SaaS exports.

## Target Customers

- Northstar Bakes
- LumaLoop Retail

## Pattern

1. Land exported data in Amazon S3.
2. Catalog with AWS Glue.
3. Query with Amazon Athena.
4. Visualize with Amazon QuickSight.
5. Automate ingestion only after manual exports prove value.

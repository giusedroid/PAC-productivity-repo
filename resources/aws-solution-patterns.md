# AWS Solution Patterns For SMB Technical Sales

## Analytics From SaaS Exports

- Trigger: Leadership wants faster reporting without replacing core systems.
- Services: S3, Glue, Athena, QuickSight.
- First proof point: One dashboard from historical exports.

## Backup And Disaster Recovery

- Trigger: Growth, acquisitions, audit findings, or operational risk.
- Services: AWS Backup, S3 Object Lock, Storage Gateway, Elastic Disaster Recovery.
- First proof point: Inventory of workloads and agreed RTO/RPO.

## IoT Telemetry Foundation

- Trigger: Field operations need earlier signals from connected assets.
- Services: IoT Core, Kinesis Data Firehose, Lambda, Timestream, S3.
- First proof point: Ingest data from one vendor or device family.

## Serverless Workflow Automation

- Trigger: Manual back-office tasks slow down small teams.
- Services: Lambda, Step Functions, EventBridge, SQS.
- First proof point: One automated workflow with human approval.

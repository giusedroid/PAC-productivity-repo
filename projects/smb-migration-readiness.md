# Project: SMB Migration Readiness

## Purpose

Create a repeatable readiness assessment for SMB customers considering AWS adoption.

## Assessment Areas

- Business trigger
- Current application landscape
- Data sensitivity
- Operational maturity
- Skills and ownership
- Budget expectations
- First workload candidate

## Output Template

- Executive summary
- Recommended first workload
- Proposed AWS services
- Risks and mitigations
- Two-week proof-of-concept plan
- Decision needed

# Task 006: Weekly Report From Repo Context

## Goal

Generate a new weekly stakeholder report from repository context.

## Read First

- `active/2026-06-weekly-focus.md`
- `customers/README.md`
- `customers/*/overview.md`
- `history/2026-06-activity-log.md`
- `projects/README.md`

## Output

Create:

- `reports/weekly/YYYY-MM-DD-stakeholder-summary.md`

Use the execution date in the filename.

## Requirements

- Include executive summary, customer movement, risks, asks, and follow-ups for next week.
- Keep it manager-friendly.
- Highlight blockers and decisions needed.
- Use only fictional repository context.

## Done When

- The report is ready for human review.
- It does not overwrite an existing report unless explicitly instructed.
